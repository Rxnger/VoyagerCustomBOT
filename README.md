# Welcome to Voyager's Custom BOT Module

To use this, please follow these steps:
1. Ensure you have Python 13+ installed.
2. Download a copy of this module.
3. Go to `settings.py` and fill in the information.
4. Open your terminal (Command Prompt, etc.) and type
   ``` pip install -r requirements.txt```
5. Once done you may run your bot and it will start to go alive.

## ⚠️WARNING: VOYAGER BOT MUST REMAIN IN THE MAIN SERVER.
